2Market Project

Data Analytics Case Study by Monica Baracho

Project Overview

The 2Market project explores customer demographics, purchase behavior, and campaign strategy. I used R, SQL, and Tableau to create dashboards and analyze patterns that help guide marketing decisions.

Key Insights

  • Identified top customer segments by income and age.
  • Analyzed advertising channel effectiveness on spend.
  • Created visual dashboards to communicate results to stakeholders.

Tools Used

R, SQL, Tableau, Excel
